Upgrading

Tahoma2D are portables so you can still keep prior versions around if you want. Just put the Tahoma2D 1.2 folder in a different place, or rename the old version's folder (example: "Tahoma2D" -> "Tahoma2D 1.1").

Transfering all your settings and configurations from v1.x to v1.2 is easy with the following steps:

  1. Open Tahoma2D v1.2
  2. Go to File -> Preferences
  3. Click on the "Import Preferences" button
  4. Select the Tahoma2D v1.x folder, where ever you placed it
  5. Select all the things you want to bring over
    - Please do not migrate Plugins
  6. Click the "Import" button
  7. Close and restart Tahoma2d 1.2
