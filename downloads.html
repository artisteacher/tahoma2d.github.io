- Windows -

  1. Click on the link to download the zipped Tahoma2D directory
  2. Right-click on the zipped file and choose "Extract All"
  3. Save the Tahoma2D directory in the location of your choice
    • DO NOT separate the Tahoma2D executable or the tahomastuff folder from the Tahoma2D directory
  4. Open the Tahoma2D directory and find the file called Tahoma2D
  5. Double-click on Tahoma2D to open

- macOS -

  1. Click on the link to download the Tahoma2D DMG file
  2. Open the DMG and drag the app to the desired location
  3. Open Tahoma2D
  4. Choose "Cancel" from the unknown developer notification dialog
  5. Right-click on Tahoma2D.app and choose "Open"
  6. Choose "Open" from the unknown developer notification dialog — afterwards Tahoma2D should open normally

For some versions of macOS, Tahoma2D must be granted permission to run for the first time in Security & Privacy, under System Preferences.

- Linux -

  1. Click on the link to download the compressed Linux appimage
  2. Unzip the appimage and place it in the desired location

- Plugins -

Windows Plugins macOS Plugins Linux Plugins
  1. Download the plugins for your OS
  2. Unzip the plugins directory
  3. Place the plugins in the tahomastuff folder's plugins directory
    • On macOS, right-click on the Tahoma2D.app and choose "Show Package Contents" to access the tahomastuff folder
  4. Open Tahoma2D and access the plugins with the special FX

For macOS only, OpenCV 4.5.1 (or a later version) must be installed on the system.
