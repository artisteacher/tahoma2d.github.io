- System Requirements -

  • OS (Windows): Microsoft Windows 8.1/10 (64-bit required)
  • OS (macOS): macOS 10.13 or a later version
  • CPU: Intel Core i series processor or better
  • RAM: 4GB or more RAM
  • HDD: 500MB or more available storage space
  • Monitor: 1280 x 1024 or higher monitor resolution

- Tahoma2D Vs. OpenToonz -

  • Tahoma2D's unique features:
  • The simplified workspace
  • Easier project management
  • Rhubarb Lip Sync integration for automatic lip syncing
  • Perspective guides and grids for precision drawing
  • A status bar with helpful hints for new users
  • No additional libraries needed to export in common formats like mp4 and gif
  • Support for Canon DSLR cameras on macOS - not just Windows
  • And many more small quality of life improvements

- FAQ -

If this is an unreported issue, create an account on Github and sign-in to start a new issue.

The more information you can add (ie. screenshots, screencaptures, crash reports, and/or repeatable steps), the more likely it is that a developer will be able to fix the problem in a timely manner.

If there is already an existing bug report for the same issue, please add more information to it instead of creating a new issue.

Create an account on Github and sign-in to submit a feature request.

Tahoma2D is portable so you can still keep prior versions around if you want. Just put the Tahoma2D 1.2 folder in a different place, or rename the old version's folder (ie: "Tahoma2D" → "Tahoma2D 1.1"). On macOS, rename the Tahoma2D app instead (ie. "Tahoma2D.app" → "Tahoma2D 1.1.app").

Transfering all your settings and configurations from v1.x to v1.2 is easy with the following steps:

  1. Open the latest version of Tahoma2D
  2. Go to File → Preferences
  3. Click on the "Import Preferences" button
  4. Select the Tahoma2D v1.x folder, wherever you placed it
  5. Select all the things you want to bring over
    • Please do not migrate Plugins
  6. Click the "Import" button
  7. Close and restart Tahoma2d 1.2
