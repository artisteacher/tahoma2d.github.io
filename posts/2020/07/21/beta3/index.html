Beta 3 is Out!

By Jeremy Bullock | July 21, 2020

Beta 3 is out now. Beta 2 accidentally didn’t ship with DSLR support, that is back in Beta 3. There are also a few bug fixes. This is also the first beta to have Mac support. Thanks to @artisteacher for all her work in helping out.

Download:

Windows: Tahoma2D Beta 3

MacOS: Tahoma2D Beta 3

The notable new features:

  • If you are using a Canon DSLR for Stop Motion, the test images now show metadata when you hover over them. Eventually I will add the ability to click a button and apply a test shot’s metadata to the current camera settings.
  • Switching projects is easier with a “Switch Project” command in the File menu.
  • Port letters and numbers have been changed in the Schematic view. Letters have been removed since they caused much more confusion than any benefit they added. Numbers now only work for actual levels. The table, camera and pegs don’t support numbers.

You can see all of the fixes and changes here.

As always, if you have any issues, please let me know. I want Tahoma2D to be as stable and user-friendly as possible.
